Handling the response object

The response to this query will return an object with 7 "name":"value" pairs. The names of each of these elements are:

  • date
  • explanation
  • hdurl
  • media_type
  • service_version
  • title
  • url

So, for example, we could slightly modify our .js file and name an html element like this:

var response = JSON.parse(req.responseText);
document.getElementById('hdPic').src = response.hdurl;

Then, we can add a line to our html file using that newly named "hdurl" element like this:

<img id="hdPic">
